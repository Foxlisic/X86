== Карта памяти DE0 ==

00000-3FFFF Быстрая память
B8000-BFFFF Видеопамять
C0000-C0FFF Память шрифтов
FC000-FFFFF БИОС

Всего использовано быстрой памяти 256+(32+4)+16=308

Видеопамять может работать в 2х режимах

- 80x25   4K,  текстовый
- 320x200 32K, графический 16 цветов
